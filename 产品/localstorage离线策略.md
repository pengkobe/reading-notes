## 考虑场景
* 数据为空 --> 立即退出
* 录入部分数据 --> 关闭App
* 数据未提交成功  --> 关闭App


```
APP 开启 -> 开启循环检测并提交
  |
  | 
  |
开始数据录入 --> 存储开始巡检时间:BeginTime 
  |
  | 
  |
单词录入 
  |
  |(save)
  |
直接缓存至本地
  |
  |
  |
退出巡检 --> 存储结束时间:EndTime 
  |
  |
  |                                                       (succeed)
组织参数提交 --> 立即存储至本地   ------->   HTTP(POST)   ---------->  删除localstorage内容
                    |                                       |
                    |(用户退出操作)                          |(fail)
                    |                                       |
                   Exit                                开启循环检测并提交                                     
```
